# 🎉 Event Management System

## 🚀 Features
- Admin Dashboard
- User Registration & Login
- Event Creation with Image Upload
- QR Code Payment System
- Payment Screenshot Upload
- Admin Approval / Rejection
- QR Code Entry Verification
- AI Assistant (Chat + Voice)

## 🛠 Tech Stack
- Node.js + Express
- PostgreSQL (Neon DB)
- HTML, CSS, JavaScript

## ⚙️ Run Project
npm install  
node server.js
